# Class: Dog
- ## Attributes:
  * weight (int)
  * name (string)
  * bread (string)
  * birthday (array)
  * known_tricks (array)

- ## Methods:
  * learn_trick (Append a trick to array known_tricks)
  * forget_trick (Removes a trick from array known_tricks)
  * current_age (subtracts birthday from current date)
  * eat_food (increases weight)
