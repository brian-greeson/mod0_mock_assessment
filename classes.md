# Class: Dog
- ## Attributes:
  * weight (int)
  * owner_name (string)
  * bread (string)
  * birthday_year (int)
  * known_tricks (array)

- ## Methods:
  * learn_trick (Append a trick to array known_tricks)
  * forget_trick (Removes a trick from array known_tricks)
  * current_age (subtracts birthday_year from current year)
  * eat_food (increases weight by 1)
